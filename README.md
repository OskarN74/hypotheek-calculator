# Hypotheek rekentool

Bereken je nieuwe maandelijkse hypotheek bedrag na extra aflossen.

<a href="https://hypotheek.oskarnijs.nl/">Demo</a>

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
npm run build
```
